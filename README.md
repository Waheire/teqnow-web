# Teqnow Web

A premium, responsive tech company website built with:

- React + Vite
- Tailwind CSS
- ShadCN-inspired reusable UI components
- Node.js + Express backend APIs

## Features

- Landing page with high-converting CTAs
- Services, Shop, About, and Contact pages
- Express contact API endpoint (`POST /api/contact`)
- Product API endpoint (`GET /api/products`)
- WhatsApp floating button
- Dark/light toggle
- Framer Motion hero animation

## Run locally

```bash
npm install
npm run dev
```

Frontend runs on `http://localhost:5173` and proxies `/api` to backend on `http://localhost:3001`.
